## Descripción del Proyecto
Este proyecto consiste en un sitio web informativo de una sola página que presenta al **Grupo SuperMegaPro**. [cite_start]El sitio ha sido construido exclusivamente con **HTML5** y **CSS3**, aplicando estructuras semánticas y un diseño basado en tarjetas para mostrar los perfiles de los integrantes[cite: 197, 198].

## Características Técnicas
[cite_start]Para cumplir con la rúbrica del laboratorio, el proyecto incluye[cite: 302, 303]:
* [cite_start]**Semántica HTML5**: Uso de etiquetas `<header>`, `<main>`, `<section>`, `<article>`, `<nav>` y `<footer>`[cite: 205].
* [cite_start]**Organización de Estilos**: Uso de un archivo CSS externo (`styles.css`) para separar la estructura de la presentación[cite: 219].
* **Identificadores y Clases**:
    * **IDs únicos**: `#equipo`, `#miembros`, `#contacto` y `#navegacion-principal`.
    * **Clases reutilizables**: `.tarjeta-estudiante`, `.texto-destacado`, `.lista-proyectos` y `.menu-lista`.
* [cite_start]**Diseño Visual**: Implementación de un layout de tarjetas con sombras, bordes redondeados y una navegación funcional mediante anclas[cite: 225, 275].
